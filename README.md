## ✨ Features

| Feature | Description |
|---------|-------------|
| 🏠 **Local Mode** | Pass one device around for secret card reveals |
| 🌐 **Online Mode** | WebRTC peer-to-peer dealing (no server) |
| 🎭 **13+ Roles** | Seer, Bodyguard, Medium, Mythomaniac, and more |
| 📖 **Narrator Guide** | Step-by-step instructions for each phase |
| ⏱️ **Discussion Timer** | 2/3/5 minute countdown with audio alerts |
| 🎨 **Mafia Edition** | Alternative theme with Mafia setting |
| 🌍 **3 Languages** | English, Spanish, Italian |
| 📱 **PWA** | Install as app, works offline |
| 🔒 **Privacy** | Zero tracking, no cookies, no analytics |

## 🚀 Quick Start

### Play Online
Visit **[n0tsosmart.github.io/slaythewerewolf](https://n0tsosmart.github.io/slaythewerewolf)**

### Install as App
- **iOS**: Safari → Share → Add to Home Screen
- **Android**: Chrome → Menu → Install app

### Run Locally
```bash
git clone https://github.com/n0tsosmart/slaythewerewolf.git
cd slaythewerewolf
npx serve  # or python -m http.server 8000
```

## 🎮 How to Play

1. **Setup** – Choose player count (5-24) and special roles
2. **Deal** – Each player views their secret card
3. **Narrate** – Follow the guide for night/day phases
4. **Eliminate** – Track deaths and votes
5. **Win** – Village wins when all wolves die, wolves win when they equal villagers

## 🛠️ Tech Stack

- Vanilla JS (ES6 modules, no framework)
- Web Components
- PeerJS for WebRTC
- LocalStorage for persistence
- Service Worker for offline
